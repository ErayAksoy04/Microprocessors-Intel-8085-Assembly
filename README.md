This repository contains a collection of Assembly Language programs written for the Intel 8085 Microprocessor. These projects were developed and tested using an 8085 Simulator as part of my Microprocessors and Computer Architecture coursework.

The primary goal of these implementations is to demonstrate a deep understanding of low-level hardware operations, memory management, and instruction set architecture (ISA).

Core Concepts Explored
Through these scripts, I implemented various low-level operations without the abstraction of high-level languages:

Bitwise Operations & Masking: Manipulating specific bits using logical instructions (AND, OR, XOR) for hardware control logic.

Memory Management: Direct memory addressing, data transfer between registers, and stack pointer manipulation.

Arithmetic & Algorithmic Logic: Implementing multi-byte addition/subtraction, loops, and conditional jumps purely at the processor level.
 Repository Structure
src/: Contains all the raw Assembly (.asm) source codes. Each file is named after the specific logical operation or algorithm it performs.

docs/: Contains the detailed laboratory reports (in PDF format). These reports explain the logic flow, register states during execution, and the final memory dumps for each project.

Environment
Language: Intel 8085 Assembly

Execution: 8085simulator

Architecture: 8-bit Microprocessor
